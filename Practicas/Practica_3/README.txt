Importate:

Las dos gramaticas se encuentran en archivos distintos:

- ari1.y para la gramatica numero 1.
- ari2.y para la gramatica numero 2.

Para poder compilar y ejecutar adecuadamente el programa, se necesita cambiar
el nombre de algun archivo de estas dos gramaticas a ari.y

Por ejemplo: 

Para compilar y ejecutar la gramatica 1 se cambia el nombre del archivo ari1.y
a ari.y

En caso de no realizar este paso, no se podra ejecutar el programa.
